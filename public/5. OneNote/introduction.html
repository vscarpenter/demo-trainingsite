Copilot for OneNote

Copilot in OneNote helps you capture, organize, and rediscover information faster. It can summarize long pages, create structured notes from rough inputs, and turn brainstorms into plans—right inside your notebooks.

Great for

  • Summarizing meeting notes into actions, decisions, and open questions
  • Transforming rough bullet lists into organized sections, tables, and tasks
  • Creating checklists, agendas, and templates from a short description
  • Surfacing key info from multiple pages in a notebook on a given topic
  • Drafting how‑to guides from scattered screenshots and snippets

Example prompts

Summarize: “From this page, produce a 6‑bullet recap with owners and due dates. Add an ‘Open questions’ list.”
Organize: “Clean up these brainstorm bullets into three sections—Goals, Risks, Next Steps—and make a checklist.”
Create: “Generate a meeting agenda for a 45‑minute status review with time boxes and a parking lot section.”
Find: “Across this notebook, pull highlights about ‘Zero Trust’ and create a summary page with links back.”

Tips

  • Context wins: Reference the current page or section and attach relevant docs when asking for summaries.
  • Be explicit: Specify structure (e.g., “use a table,” “create a checklist,” “3 sections with headings”).
  • Iterate: Ask Copilot to list assumptions; follow up with Refine to correct, add detail, or change tone.
  • Link back: Request links to original pages so others can verify source material quickly.

Common scenarios

  • Meeting capture → summary + tasks posted to Planner/To Do
  • Project notebook cleanup → create a “Start here” page with quick links
  • Research roundup → pull quotes and references into a single outline
  • Runbooks → convert screenshots + steps into a repeatable checklist

Review outputs for accuracy and sensitive content before sharing.
